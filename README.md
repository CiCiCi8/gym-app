# Gym App

Spring Boot aplikacija za upravljanje članovima teretane.

## Funkcionalnosti
- Dohvatanje svih članova
- Dodavanje novog člana
- Povezivanje sa PostgreSQL bazom
- Testiranje API endpointa preko Postmana

## Endpointi
- GET `/getmembers`
- POST `/addmember`

## Tehnologije
- Java
- Spring Boot
- PostgreSQL
- JPA / Hibernate
- Postman
- Git / GitHub
